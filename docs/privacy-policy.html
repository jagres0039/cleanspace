Kebijakan Privasi CleanSpace

Terakhir diperbarui: 22 Juni 2026

CleanSpace (“aplikasi”, “kami”) adalah aplikasi pembersih penyimpanan Android. Kebijakan ini menjelaskan data apa yang diakses aplikasi, bagaimana digunakan, dan hak Anda. Dengan memakai CleanSpace, Anda menyetujui kebijakan ini.

1. Data yang diakses aplikasi

  • File & media di perangkat (foto, video, audio, dokumen, cache, folder tersembunyi) — dipindai hanya di perangkat Anda untuk menemukan file besar, duplikat, dan sampah. Isi file tidak pernah diunggah ke server kami atau pihak mana pun.
  • Daftar aplikasi terpasang & statistik penggunaan (Usage Access) — untuk menampilkan ukuran cache dan aplikasi yang jarang dipakai. Diproses lokal, tidak dikirim ke mana pun.
  • Advertising ID (AAID) — digunakan oleh Google AdMob untuk menampilkan iklan. Lihat bagian Iklan.

2. Kami tidak mengumpulkan data pribadi ke server

CleanSpace tidak memiliki server backend untuk menyimpan file atau data pribadi Anda. Semua pemindaian dan penghapusan berjalan sepenuhnya di perangkat. Kami tidak menjual atau menyewakan data Anda kepada siapa pun.

3. Penghapusan file

Penghapusan file selalu memerlukan konfirmasi Anda. Untuk membersihkan cache/data aplikasi lain atau meng-uninstall aplikasi, CleanSpace mengarahkan Anda ke dialog resmi Android — kami tidak menghapus data aplikasi lain secara diam-diam.

4. Iklan (Google AdMob)

Aplikasi menampilkan iklan melalui Google AdMob. AdMob dapat menggunakan Advertising ID dan data perangkat untuk menayangkan iklan, termasuk iklan yang dipersonalisasi. Untuk pengguna di wilayah EEA/Inggris, kami menampilkan formulir persetujuan (Google UMP) sebelum iklan dipersonalisasi diaktifkan.

Pelajari cara Google memakai data: policies.google.com/technologies/partner-sites.

5. Izin yang diminta

  • READ_MEDIA_* / READ_EXTERNAL_STORAGE / MANAGE_EXTERNAL_STORAGE — memindai dan membersihkan file.
  • PACKAGE_USAGE_STATS — ukuran cache & aplikasi jarang dipakai.
  • QUERY_ALL_PACKAGES — menampilkan daftar aplikasi terpasang.
  • INTERNET / AD_ID — memuat iklan.

6. Anak-anak

CleanSpace tidak ditujukan untuk anak di bawah 13 tahun dan tidak secara sengaja mengumpulkan data dari mereka.

7. Perubahan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal pembaruan terbaru.
